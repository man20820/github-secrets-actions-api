# Github Secrets Actions API

Allow you to bulk input actions -> secrets

## Install
1. set dot env
2. copy your secrets to .secrets
3. copy your variables to .vars
4. npm install

## Github Secrets
### Upsert
```bash
node upsert.js
```

### Clear
```bash
node clear.js
```

## Github Variables
### Insert
```bash
node varsInsert.js
```

### Clear
```bash
node varsClear.js
```
